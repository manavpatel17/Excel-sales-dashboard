📊 Excel Sales Dashboard

🔍 About This Project

This is a simple Excel project where I worked on a sales dataset and tried to analyze it like a real business problem.

In this project, I cleaned messy data, used pivot tables for analysis, and created a dashboard to show insights in a visual way.

🧹 What I Did in Data Cleaning
Removed duplicate rows
Fixed inconsistent names (city, product, etc.)
Handled missing values
Converted dates into proper format

📊 Data Analysis
Used Pivot Tables to analyze:
Sales by city
Sales by category
Customer performance

Found:
Top city with highest sales
Best customer
Most selling product

📈 Dashboard
Created charts to visualize data
Used filters (slicers) to make it interactive
Designed a simple dashboard layout

🧠 What I Learned
How to clean real-world messy data
How to use pivot tables effectively
How to present data using charts and dashboards
